=== AI-DRIVEN RESUME SCREENING - QUICK START (Windows/PowerShell) ===

# Open four PowerShell terminals, cd to the project root, then run these:

TERMINAL 1 - Backend (FastAPI):
cd backend
./venv/Scripts/Activate.ps1
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000

TERMINAL 2 - Frontend (React):
cd frontend/c
npm start

TERMINAL 3 - Rasa Actions:
cd chatbot
./venv/Scripts/Activate.ps1
rasa run actions --port 5055

TERMINAL 4 - Rasa Server:
cd chatbot
./venv/Scripts/Activate.ps1
rasa run --enable-api --cors "*" --port 5005

=== URLS ===
Backend API: http://localhost:80009
Frontend: http://localhost:3000
Rasa Actions: http://localhost:5055
Rasa Server: http://localhost:5005
API Docs (Swagger): http://localhost:8000/docs

=== Notes ===
- Run from the project root (the folder that contains backend/, frontend/, chatbot/)
- Keep all four terminals open while testing.
- If virtual envs or node_modules are missing, see HANDOFF.md for one-time setup steps.
